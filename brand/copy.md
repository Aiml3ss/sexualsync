# Sexualsync · Brand Copy

All approved copy lives here. Use exactly as written.

---

## Brand name

**Sexualsync** — set as the wordmark in two parts joined by a rose dot:

> *sexual* · *sync*

The wordmark is **always italic**, **always lowercase** when set as the mark, in **Cormorant Garamond** at weight 500. "sexual" is cream, "sync" is rose. The dot between them is a small rose circle (not a typographic middot — see `wordmark/wordmark.svg`).

When written in body copy: **Sexualsync** — title-cased, one word, no spaces.

---

## Tagline

> **Get curious. Get in sync.**

- Two short imperatives. Always together, always in that order.
- When styled for display, "Get in sync." takes the rose accent color.
- Use the period after each sentence — the cadence matters.

**Do not** abbreviate to "Get in sync" alone, and **do not** reorder.

## Description (primary — "direct medium")

Use this everywhere a paragraph of explanation is needed: landing page hero, App Store description, share-card subtitle, partner-invite body.

> Some things are easier to type than say. Sexualsync is a private space for couples to send requests, share fantasies, and find each other's yes — without the awkward pause.

**Word count:** 33. **Reading level:** plain. Three commas, one em-dash. Don't rewrite for "polish" — the rhythm is intentional.

## Voice rules

1. **Type, don't talk.** The product exists because some things are easier written. The copy should feel like it understands that — slightly hushed, never crude or "spicy."
2. **Two, not many.** Always "two of you," "for two," "a pair." Never "users," "community," "members."
3. **Maybe is a complete sentence.** Curiosity wins over performance. We don't sell intensity; we sell the conversation.
4. **No emoji except where the product already uses them** (the in-app `Ask` chip icons). Marketing surfaces stay typographic.
5. **No exclamation marks.** Periods only. Trust the words.
